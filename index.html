CRUD de MVC en PHP con SQL y Bootstrap

Este archivo es parte de un sistema CRUD desarrollado utilizando el patrón Modelo-Vista-Controlador (MVC) con PHP y una base de datos SQL. Además, se ha utilizado Bootstrap para el diseño y la interfaz de usuario.

Funcionalidades incluidas:

  • Creación, lectura, actualización y eliminación (CRUD) de registros de vehículos.
  • Formulario de actualización de vehículos con validación de datos.
  • Listado de vehículos con opciones de edición y eliminación.

Tecnologías utilizadas:

  • PHP: Lenguaje de programación del lado del servidor.
  • SQL: Lenguaje de consulta estructurada para la gestión de la base de datos.
  • Bootstrap: Marco de diseño para la creación de interfaces de usuario.

Uso

Este archivo forma parte de un proyecto más grande y se utiliza para proporcionar una visión general del sistema CRUD desarrollado. Para utilizar el sistema completo, asegúrate de tener acceso a la base de datos y los archivos PHP correspondientes.
